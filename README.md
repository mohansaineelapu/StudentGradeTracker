Student Grade Tracker

Project Description

Student Grade Tracker is a Java-based GUI application developed using Java Swing. The application allows users to enter student names and grades, store the information, and generate a summary report showing average, highest, and lowest scores.

Features

- Add student names and grades
- Store data using ArrayList
- Calculate average score
- Find highest score
- Find lowest score
- Display student records
- User-friendly graphical interface using Java Swing

Technologies Used

- Java
- Java Swing
- ArrayList
- Object-Oriented Programming (OOP)

Project Structure

- StudentGradeTrackerGUI.java

How It Works

1. Enter the student's name.
2. Enter the student's grade.
3. Click Add Student.
4. Repeat for multiple students.
5. Click Show Summary to view:
   - Average Score
   - Highest Score
   - Lowest Score

Requirements

- JDK 8 or above
- VS Code (or any Java IDE)

How to Run

Compile the program:

javac StudentGradeTrackerGUI.java

Run the program:

java StudentGradeTrackerGUI

Sample Output

- Student Name: Mohan
- Grade: 85

Summary:

- Average Score: 85.0
- Highest Score: 92.0
- Lowest Score: 78.0
